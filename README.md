# Simple Linux System Info / Monitoring Project

## Project Overview
This is a beginner-friendly DevOps project created using Linux shell scripting.  
The script collects basic system information and monitoring details.

## Features
- Displays system information
- Checks disk usage
- Generates system report log
- Uses Bash shell scripting
- Beginner DevOps project for GitHub portfolio

## Tools Used
- Linux / Ubuntu WSL
- Bash Script
- Git
- GitHub

## How to Run

```bash
chmod +x monitor.sh
./monitor.sh
